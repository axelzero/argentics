Описание проекта:
В проекте есть простая сцена (Argentics/Scenes/Level), которая является адаптацией 2DCharacter из стандартных ассетов в 3D. 
Сейчас там есть простой игрок, несколько платформ, некоторые объекты и выход из уровня.

Что нужно сделать:
 - создать возможность игроку умереть. Для этого нужно расширить класс PlatformerCharacter, добавив в него жизни(пусть их будет 3); 
	скачать любую mecanim анимацию смерти и добавить её к персонажу; 
	когда жизни кончаются, проигрывается анимация смерти персонажа, 				
	затем пару секунд анимации в UI (на своё усмотрение можно создать любую анимацию) и уровень перезапускается.

 - добавить в игру один-два вида ловушек и расставить их по уровню, на свой вкус. При соприкосновении с игроком ловушка наносит ему урон. 

 - добавить в игру одного врага, который стреляет в игрока, когда тот находится достаточно близко; снаряд должен лететь медленно, 
	чтобы игрок мог от него увернуться. выбор моделей и эффектов на свой вкус.

 - сейчас уровень построен полностью из примитивов. Нужно найти любые бесплатные ассеты на свой вкус и вдохнуть в этот уровень жизнь - добавить скайбокс, 
	заменить платформы(используется новая система префабов), можно добавить пропсы, пост-эффекты и поиграть с освещением.

 - уровень нужно немного увеличить, чтобы прохождение занимало не менее 40 секунд

 - сейчас в игре есть баги, которые нужно самостоятельно найти и починить

 - игра должна собираться на android, уровень должен быть проходим.
